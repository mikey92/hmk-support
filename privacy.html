개인정보처리방침

시행일: 2025년 1월 1일 | 최종 수정: 2026년 3월 11일

HMK 자산관리(이하 "서비스")는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」을 준수합니다. 본 개인정보처리방침은 서비스가 수집하는 개인정보의 항목, 수집 목적, 보유 기간, 이용자의 권리 등을 안내합니다.

1. 수집하는 개인정보

수집 시점 수집 항목 수집 방법
회원가입(Google 로그인) 이름, 이메일 주소, Google 계정 ID Google Sign-In API
서비스 이용 자산 정보, 투자 포트폴리오, 일정 데이터, 가구 구성원 정보 이용자 직접 입력
자동 수집 기기 정보(OS 버전, 앱 버전), 서비스 이용 기록 자동 수집

2. 개인정보의 처리 목적

  • 회원 관리: 본인 확인, 로그인 인증, 가구 구성원 관리
  • 서비스 제공: 자산 데이터 저장 및 기기 간 동기화, AI 투자 조언 제공, 캘린더 관리
  • 서비스 개선: 이용 통계 분석, 오류 진단

3. 개인정보의 보유 및 파기

  • 회원 탈퇴 시 지체 없이 파기합니다.
  • 단, 관련 법령에 의해 보존이 필요한 경우 해당 기간 동안 보관합니다.
  • 1년 이상 미접속 계정은 별도 저장 후 파기합니다.

4. 개인정보의 제3자 제공

서비스는 이용자의 동의 없이 개인정보를 제3자에게 제공하지 않습니다. 다만, 서비스 운영을 위해 아래와 같이 위탁합니다.

위탁 업체 위탁 업무 제공 항목
Google (Firebase) 데이터 저장, 인증, 동기화 이메일, 이름, 자산 데이터
Google (Sign-In) 소셜 로그인 인증 Google 계정 정보
OpenAI (ChatGPT) AI 투자 조언 제공 자산 데이터 (익명화 처리)
Brave Search 웹 검색 기능 검색어

5. 이용자의 권리

  • 개인정보 열람, 정정, 삭제를 요청할 수 있습니다.
  • 앱 내 설정에서 직접 계정을 삭제할 수 있습니다.
  • 개인정보 처리 정지를 요구할 수 있습니다.
  • 위 권리 행사는 아래 연락처로 요청할 수 있습니다.

6. 개인정보의 안전성 확보 조치

  • Firebase 보안 규칙을 통한 데이터 접근 제어
  • HTTPS/TLS를 통한 데이터 전송 암호화
  • Apple Keychain을 통한 인증 토큰 안전 보관
  • 가구 단위 데이터 격리 (다른 가구의 데이터에 접근 불가)
